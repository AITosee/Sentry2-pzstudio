#### 安装教程

1.  在下方相关链接中下载PZstudio程序安装包
2.  下载本项目到PC端并解压
3.  安装完成后选择帮助->加载自定义组件->选择上一步解压出来的文件中的dsx.json配置文件，即可将大师兄组件加载到PZstudio中
